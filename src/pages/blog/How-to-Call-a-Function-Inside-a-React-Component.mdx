    ### How Do You Call a Function Inside a React Component

    We will define a function that will log to the console "HELLO WORLD" inside a react child component.

    ( 1. ) Import the `useRef` hook from React in the parent component and invoke it.

    ```js
    import { useRef } from 'react'

    ```
    <br />


    ( 2. ) Pass the `childRef` as a prop to the child component.

    ```js
    function ParentComponent() {
        const childRef = useRef(null);

        return (
            <div>
            <ChildComponent childRef={childRef} />
            <button onClick={() => childRef.current.childFunction()}>Call Child Function</button>
            </div>
        );
    }
    ```
    <br />

    ( 3. ) In the child component, receive the `childRef` prop as a parameter.

    ```js
        function ChildComponent(props) {
            // Extract the `childRef` from the props object.
            const { childRef } = props;

        // ...
        }
    ```
    <br />

    ( 4. ) Import the `useEffect` hook from React in this child component and invoke it. Remember to pass an empty dependency array as the second argument after the arrow function.

    ```js
    import { useEffect } from 'react'

    function ChildComponent(props) {
        // Extract the `childRef` from the props object.
        const { childRef } = props;

        useEffect(() => { }, []);

        // ...
    }

    ```
    <br />

    ( 5. ) Within the 'useEffect' hook, assign an anonymous function to the 'current' property of the 'childRef' object.

     ```js
    import { useEffect } from 'react'

    function ChildComponent(props) {
        // Extract the `childRef` from the props object.
        const { childRef } = props;

        const anonymousFunc = () => {
                // Define the logic of the function
                console.log("HELLO WORLD")
        };

        useEffect(() => {
            childRef.current.childFunction = anonymousFunc
        }, []);

        // ...
    }

    ```
    <br />

    ( 6. ) Back in the parent component file, you can now call the `function` defined in this child component using the `childRef` object. Pay attention to the `handleButtonClick` function definition.

    ```js
    import { useRef } from 'react'

    function ParentComponent() {
        const childRef = useRef(null);

        const handleButtonClick = () => {
            // Call the child component's function
            childRef.current.childFunction();
        };

        return (
            <div>
            <ChildComponent childRef={childRef} />
            <button onClick={handleButtonClick}>Call Child Function</button>
            </div>
        );
    }

    ```
